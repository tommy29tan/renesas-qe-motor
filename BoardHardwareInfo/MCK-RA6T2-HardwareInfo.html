MCK-RA6T2 Renesas Flexible Motor Control Kit for RA6T2 MCU Group

The MCK-RA6T2 is a development kit that enables easy evaluation of motor control using permanent magnet synchronous motors (brushless DC motors). With this product and the sample code and QE for Motor that can be downloaded from the website, you can start evaluating motor control using the RA6T2 right away.

Hardware Information

  • The CPU board is equipped with the RX26T as the motor control MCU.
  • Inverter board for 3-phase BLDC motor (48V/10A rating)
  • On-board debugger for flash programming of RX26T
  • Supports 1-shunt/3-shunt current sensing
  • Over current detection
  • Supports the motor control development support tool "Renesas Motor Workbench"
  • Motor control evaluation and debugging can be performed safely with the communication board which isolates PC and the CPU board electrically
  • Included items
    • RX26T CPU board (MCB-RX26T Type A)
    • Inverter board (MCI-LV-1)
    • Communication board (MC-COM)
    • Permanent magnet synchronous motors
    • Accessories (cables, standoffs, etc.)
